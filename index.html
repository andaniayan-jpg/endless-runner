How To Play

Run as far as possible while avoiding rocks, spikes, crates and logs.

Collect coins to increase your score, and grab power-ups to survive longer.

Speed increases continuously, making the game more difficult over time.

About This Project

I built RunnerRush to practice core front-end game development without relying on any external engine or library. Everything you see is drawn frame by frame on the HTML5 Canvas.

Skills and tools I practiced:

  • HTML5 Canvas 2D rendering (shapes, gradients, transforms, animation frames)
  • Vanilla JavaScript game loops using requestAnimationFrame
  • Game physics: gravity, velocity, collision detection
  • State management (game states, pause/resume, difficulty modes)
  • Procedural spawning and difficulty scaling
  • The Web Audio API for generating sound effects without audio files
  • Responsive CSS layout and mobile touch/swipe controls
  • Browser localStorage for persisting high scores
  • Debugging real bugs: CSS cascade order, event handling, and DOM/canvas overlays
