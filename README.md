# endless-runner

**endlessrun is a simple html and jawascript game, where player can play this game endlessly and enjoy their high score eerytime!**

# Why making endless-runner

**AI was used only for functions and domain names, the logic, math and design is my own**

**I wanted to learn how to make the canvas with jawa-script, after making simple games with html and three.js, i wanted to learn how to make an endless canvas with random circumstances throughout the enivronment. I also learned different functions like collision detection and coin collection**

# How to play

- Visit this link to play the game =>  https://endless-runner-orpin.vercel.app

- Press start to play, obstacles with start comming once the game starts

- Press SPACE to jump upon the obstacles. 

- To slide, use ARROW DOWN.

- Press P, R, M for Pause, Restart, and Mute

- Collect the coins and increase your high school!
